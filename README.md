# React Movie Fetcher App

A simple **Movie Fetcher application** built with **React**. Search for movies and get details like title, poster, and rating using an external API.

---

## Features

- Search movies by title
- View movie details (poster, overview)
- Responsive and interactive UI
- Handles empty or invalid search results gracefully

---

## Technologies Used

- React
- JavaScript (ES6+)
- HTML & CSS
- TMDb API
